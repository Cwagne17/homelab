# Setup Overview

Get your environment ready to deploy the homelab infrastructure.

## What You Need

The homelab requires:

- **Proxmox VE Server**: Virtualization host (physical or nested)
- **Development Machine**: Where you'll run Atmos, Packer, and OpenTofu
- **Network Access**: Connectivity between your machine and Proxmox
- **Basic Knowledge**: Familiarity with Linux, VMs, and Kubernetes concepts

## Setup Steps

### 1. Prerequisites

Ensure you have the required hardware and software:

- [Prerequisites Guide](prerequisites.md)

### 2. Install Tools

Install Atmos, Packer, OpenTofu, and kubectl:

- [Installation Guide](installation.md)

### 3. Configure Proxmox

Set up API access and networking:

- [Proxmox Setup](../proxmox/setup.md)

## Estimated Time

- **First-time setup**: 1-2 hours
- **Subsequent deployments**: 15-30 minutes
